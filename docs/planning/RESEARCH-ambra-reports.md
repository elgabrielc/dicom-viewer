# Research: Ambra Health Document/Report Handling

## Research Question

How does Ambra Health handle document attachments and reports in their cloud PACS platform?

## Specific Areas to Investigate

1. **Document Upload**: How do users attach reports/documents to imaging studies in Ambra?

2. **Supported File Types**: What formats are supported beyond DICOM?
   - PDF reports
   - Images
   - Other document types

3. **Storage and Persistence**: How does Ambra store uploaded documents?
   - Cloud architecture
   - Integration with DICOM studies
   - Retention policies

4. **Viewing Experience**: How are documents displayed?
   - In-viewer vs separate window
   - Mobile experience
   - Integration with image viewing workflow

5. **API/Integration**: How do external systems interact with Ambra's document features?
   - Upload APIs
   - Retrieval methods
   - FHIR/HL7 integration

## Context

Ambra Health is a cloud PACS platform. Understanding their document handling will provide insight into professional-grade medical document management, particularly for cloud-based solutions.
